# Tugas Pelatihan Version Control System 2026

## Pre-requisite

Lakukan fork pada repository ini. Setelah itu, lakukan clone pada repository hasil fork tersebut di local Anda.

- https://github.com/Lab-RPL-ITS/module-pelatihan-vcs

## Deskripsi Tugas

1. Buatlah sebuah **branch baru** dengan format `penugasan/nama_anda`.

2. Buatlah sebuah **folder** dengan nama `nama_anda` (contoh: `rpl`) pada folder `tugas/pelatihan-vcs-2026/`.

3. Salin **file** yang ada pada folder `landing-page` ke folder `/tugas/pelatihan-vcs-2026/nama_anda` yang telah dibuat.
   Contoh: `/tugas/pelatihan-vcs-2026/rpl`.

4. Tambahkan sebuah file `README.md` di dalam folder Anda yang berisi:

   ```
   Nama:
   Pelatihan: VCS RPL 2026
   ```

5. Buatlah sebuah **commit** dengan pesan sesuai dengan pesan commit yang telah dijelaskan pada materi.

6. Push ke remote repository.

7. Setelah itu, buat Pull Request dari branch `penugasan/nama_anda` di repository fork Anda menuju branch `main` pada repository pelatihan `(Lab-RPL-ITS/module-pelatihan-vcs)`.

**Note**

- Tugas ini merupakan tugas individu yang merupakan syarat mendapatkan sertifikat pelatihan VCS RPL 2026
- **Jangan mengubah file pada folder `landing-page`**, gunakan folder tersebut hanya sebagai template
- Pastikan seluruh pekerjaan dilakukan di folder `tugas/pelatihan-vcs-2026/nama_anda`
- Apabila terdapat pertanyaan mengenai tugas dan sebagainya bisa langsung membuat pertanyaan pada:
  https://github.com/Lab-RPL-ITS/module-pelatihan-vcs/discussions

## Deadline
